# 🚀 My Portfolio

This is a simple and modern personal portfolio built using **HTML + Tailwind CSS**.  
It’s designed to showcase my skills, projects, and contact information in a clean and responsive way.

---

## 🌟 Features

- Clean and modern UI design
- Built with Tailwind CSS (utility-first framework)
- Fully responsive layout (mobile friendly)
- Simple and fast loading
- Easy to customize and expand

---

## 🛠️ Built With

- HTML5
- Tailwind CSS (CDN)

---

## 📸 Preview

You can open the project in any browser to view the portfolio locally.

---

## 🚀 How to Use

1. Download or clone the repository
2. Open `index.html` in your browser
3. Done 🎉
